# Quantifying-the-Impact-of-Exchange-Rate-Volatility-on-Key-Economic-Sectors-in-Sri-Lanka-
📌 Introduction

This project focuses on analyzing how exchange rate fluctuations (LKR/USD) affect key economic sectors in Sri Lanka. As a developing country, Sri Lanka heavily depends on imports, exports, and tourism, making exchange rate movements an important factor in economic performance.

The purpose of this study is to understand how changes in the exchange rate influence these sectors using real-world data and statistical analysis techniques.

🎯 Objectives

The main objectives of this project are:

To analyze the relationship between exchange rate and imports
To examine how exchange rate changes affect exports
To study the impact on tourism earnings
To apply statistical and time series methods to identify trends and patterns

📁 Data Collection

The data used in this project were collected from reliable sources such as:

Central Bank of Sri Lanka
Department of Census and Statistics

The dataset includes:

Exchange rate (LKR/USD)
Interest rates (Treasury bill rates)
Inflation rate
GDP
Imports and exports (USD)
Tourism earnings

Some data were available in different formats (weekly, annual), so they were converted into monthly values for consistency.

⚙️ Data Preprocessing

Before analysis, several steps were taken to prepare the data:

Missing values were handled using interpolation
Outliers were identified using boxplots
Normality was checked using probability plots
Log transformation was applied where necessary

Even after transformation, some variables still had outliers, so both original and transformed data were considered in analysis.

🧠 Methodology

This project uses a combination of statistical and time series methods:

Descriptive statistics to understand data behavior
Correlation analysis to identify relationships
Multiple regression to measure impact of variables
Time series analysis to study trends
Moving average and Winters’ method for forecasting
📈 Key Findings
Exchange rate has a noticeable impact on imports and exports
Imports tend to decrease when exchange rate increases
Exports show a positive relationship with exchange rate
Tourism earnings are less directly affected and depend more on overall economic conditions
Regression models show moderate explanatory power due to real-world complexity
🔮 Forecasting

Time series analysis was used to forecast future values for imports, exports, and tourism earnings.

Moving average method provided the best results based on error measures (MAPE, MAD)
Short-term forecasts (4 months ahead) show relatively stable trends
